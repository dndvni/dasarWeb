Position: static;

Position static digunakan untuk mengatur element menjadi statis secara default. elemen akan mengikuti posisi normal secara default, elemen tidak dipengaruhi ileh properti seperti top, bottom, left dan right.:

elemen ini diatur menggunakan posisi static;




Position: relative;

sebuah element HTML yang menggunakan position relative akan terletak pada posisi normal. mengatur properti atas, kanan, bawah, dan kiri dari elemen yang diposisikan dengan "relatif" akan membuatnya jauh dari posisi normalnya. konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalakan oleh elemen tersebut.:

elemen ini diatur menggunakan posisi relative;



Posistion: fixed;

sebuah elemen HTML yang di setting dengan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman website di scrooll. berlaku pengaturan left, bottom, top dan right pada posistin fixed:

elemen ini diatur menggunakan posisi fixed;



Position: absolute;

Elemnt HTML yang menggunakan osition absolute akan diposisikan relatif dengan element lain yang mendahuluinya yang terdekat, bukan relatif terhadapa lyar secara normal.:

Elemen ini diatur menggunakan posisi: relative;
elemen ini diatur menggunakan posisi: absolute;
